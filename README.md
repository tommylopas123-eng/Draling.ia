# Draling — Automatización con IA para empresas

> Repositorio central de la empresa **Draling**. Acá vive toda la información:
> visión, estrategia, marketing, materiales de venta y demos.

## 📂 Qué hay en este repo

| Archivo / carpeta | Qué es |
|-------------------|--------|
| `VISION.md` | La visión de la empresa (fuente de verdad) |
| `PLAN.md` | Plan maestro con checklist (qué está hecho y qué falta) |
| `ESTUDIO-MERCADO.md` | Estudio de mercado: demanda, competencia, segmentos y precios |
| `PROSPECTOS.csv` | Lista de prospección: a quiénes contactar y con qué mensaje |
| `PLAN-REDES.md` | Plan de redes (Instagram, TikTok, LinkedIn) con calendario y guiones |
| `LANDING.md` | Copy de la landing page |
| `landing/` | Landing page lista para publicar (HTML) |
| `demo-bot/` | Demo interactiva del bot de WhatsApp (catálogo real) |
| `APRENDER-AUTOMATIZACION.md` | Ruta de 8 semanas para aprender a automatizar |
| `pdf/` | Versiones en PDF de los documentos |
| `.agents/product-marketing.md` | Contexto de marketing (lo usan las skills de Claude) |

## 🏢 La empresa en una frase

Draling automatiza las tareas repetitivas de pymes y startups con IA:
WhatsApp que responde solo, pedidos que se cargan solos, reportes automáticos.

**Estrategia en dos fases:** primero servicio/agencia de automatización
(facturar + aprender), después la plataforma todo-en-uno.
